# 휴먼커넥트에듀센터 홈페이지

## GitHub Pages에 올리는 방법 (아주 간단해요)

1. 이 zip 파일 압축을 풉니다.
2. 압축을 풀면 나오는 **index.html** 파일 하나만 사용하시면 됩니다.
   (이 파일 안에 디자인·이미지·기능이 전부 들어있어서, 이 파일 하나만 있으면 홈페이지가 완성돼요.)
3. GitHub 저장소(`gm751020.github.io` 안의 `hcec` 폴더 등)로 이동합니다.
4. 기존에 있던 index.html을 삭제합니다.
5. 방금 받은 index.html을 업로드합니다. (파일 이름이 정확히 `index.html`이어야 해요)
6. Commit 하면 몇 분 안에 실제 홈페이지에 반영됩니다.

폴더 구조 신경 쓰실 필요 전혀 없어요 — 파일 하나만 잘 올리면 끝입니다.

## source 폴더는 무엇인가요?

`source` 폴더 안에는 나중에 수정할 때 참고할 수 있도록 원본 코드(HTML, CSS, JS, 이미지 파일)를 각각 나눠서 넣어뒀어요.
지금 당장은 사용하지 않으셔도 되고, 나중에 개발자에게 수정을 맡기거나 저와 다시 작업할 때 필요하시면 참고하시면 됩니다.

- source/index.html — 페이지 구조
- source/css/style.css — 디자인
- source/js/script.js — 동작(메뉴, 폼, 애니메이션 등)
- source/assets/ — 로고 이미지 원본
